# 资源压缩和上传工具

一个基于Python和Tkinter的GUI工具，用于7z分卷压缩和百度网盘上传。

## 功能特性

1. **文件夹压缩**
   - 支持浏览选择文件夹
   - 支持7z分卷压缩
   - 可自定义解压密码
   - 可根据文件大小调整分卷大小

2. **自动生成说明文本**
   - 自动生成包含文件列表和大小的说明文本
   - 可直接复制到剪贴板

3. **百度网盘上传**
   - 自动生成上传目录
   - 支持自定义上传目录
   - 自动获取分享链接和提取码

4. **设置选项**
   - 可自定义压缩文件临时保存位置

## 依赖项

- Python 3.6+
- 7-Zip (需要安装并添加到系统PATH)
- BaiduPCS-Go (百度网盘命令行工具，需要安装并添加到系统PATH)

## 安装和使用

1. **安装依赖**
   - 安装7-Zip: 从官方网站下载并安装
   - 安装BaiduPCS-Go: 从GitHub下载并添加到系统PATH
   - 安装Python 3.6+ (如果尚未安装)

2. **运行程序**
   ```bash
   python main.py
   ```

3. **使用步骤**
   - 在"压缩"选项卡中选择文件夹
   - 设置解压密码和分卷大小
   - 点击"开始压缩"按钮
   - 压缩完成后，会自动生成说明文本
   - 在"上传"选项卡中点击"上传到百度网盘"
   - 上传完成后，会显示分享链接和提取码

## 注意事项

- 确保7-Zip和BaiduPCS-Go已正确安装并添加到系统PATH
- 使用BaiduPCS-Go前需要先登录百度网盘
- 大型文件压缩和上传可能需要较长时间
- 压缩文件会保存在设置的临时目录中
